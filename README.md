# Modern React Todo Architecture 🚀

A professional-grade Todo application built with **React 19** and **Vite**, focusing on clean code and scalable architecture.

## 🛠 Features
- **CRUD Operations**: Create, Read, Update (Toggle), and Delete tasks.
- **Custom Hooks**: Business logic is separated into `useTodoLogic.js` for better maintainability.
- **Component-Driven Design**: UI is broken into modular, reusable components.
- **State Persistence**: (Optional: Add this next!) Tasks stay even after page refresh.

## 🧠 What I Learned
- **Lifting State Up**: Managing data flow from child components to a central parent.
- **Immutability**: Using `.map()` and `.filter()` to update state without mutating the original array.
- **Prop Drilling vs. Clean Props**: Passing functions as props to handle user actions.
